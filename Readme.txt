- Install dotnet core vesion 2.2. 
	- Instructions for all supported OS are here: https://dotnet.microsoft.com/download#linuxubuntu
	- Make sure you change the version to 2.2 where needed.
- For building the code:
	# dotnet build
- For testing the code:
	# dotnet test
- For running the code:
	# dotnet run --project MagicSquare

- Code was built, tested and run against Ubuntu 18.04, dotnet core version 2.2
- Entry point (Main function) is in MagicSquare/Program.cs
